✅ ToDo List App

A ToDo List web application built with HTML, CSS and JavaScript.

This project allows users to add, edit, delete and save tasks directly in the browser.

🌐 Live Demo

https://usmrem.github.io/ToDOList/


🚀 About The Project

This project was created to practice JavaScript and DOM manipulation by building an interactive task management application.

The app allows users to manage their tasks and keep them saved in the browser using localStorage.


🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* localStorage


📂 Features

* Add new tasks
* Edit tasks
* Delete tasks
* Save tasks in localStorage
* Tasks remain after page refresh
* Clean and responsive interface


🔧 Future Improvements

* Add task completion checkbox
* Add task categories
* Add dark mode
* Improve UI design
